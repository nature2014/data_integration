## DP data protocol message   
*  提供统一的数据多协议支持，采用spring-integration来支持多协议接口调用支持。  
*  提供Hessian作为高效地二进制协议来实现RPC调用。  

[Spring integration site](http://projects.spring.io/spring-integration/) 
[Hessian site](http://hessian.caucho.com/)
